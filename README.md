## Cara Install Project Repositori

### install composer

```bash
    composer install
    atau
    composer update
```

### ganti nama databasenya

```bash
    DB_DATABASE=(nama database)
```

### migrate database

```bash
    php artisan migrate
```

### install package laravel ui

```bash
    DB_DATABASE=(nama database)
```

### jalankan project untuk test dan serve

```bash
    php artisan test
    php artisan serve

```
